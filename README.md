## 🎯 두 갈래로 사용하기

### 방법 A — 즉시 사용 (간단)

이미 만들어둔 `kings/` 폴더를 그대로 쓰세요.

1. `kings/` 폴더를 GitHub에 업로드
   ```
   cd kings
   git init
   git add .
   git commit -m "init: 조선 27왕 골격 생성"
   gh repo create jung-youngsu/joseon-27kings --public --source=. --push
   ```
   (GitHub CLI가 없으면 GitHub 웹에서 새 레포 만들고 `git remote add origin ...` 해서 push)

2. VS Code에서 폴더 열고 작업 시작
3. EP1 이성계편 작업물을 `ep01_taejo/` 폴더로 옮기기
4. 이방과편 만들 때는 `ep02_jeongjong/` 폴더에서 작업

### 방법 B — 직접 실행 (커스터마이징)

`generate-kings.ps1`을 PowerShell에서 실행하면 같은 결과물이 다시 만들어져요.

1. `kings.json`과 `generate-kings.ps1`을 원하는 위치에 두기
2. VS Code 터미널에서:
   ```powershell
   .\generate-kings.ps1
   ```
3. 결과물이 같은 폴더 아래 `kings/`에 생성됨

> ⚠️ 이미 있는 파일은 덮어쓰지 않아요. 안전합니다.

`kings.json`을 수정하면 (예: tagline 변경, ep 추가) 스크립트를 다시 돌려서 반영할 수 있어요.

## 🔧 HTML → JSON 추출기 사용법

`extractor.html`을 더블클릭해서 브라우저에서 열어요.

### 이성계편 다듬기 시나리오

1. **이성계편 HTML 파일을 좌측 드롭존에 드롭**
   (또는 클릭해서 파일 선택)

2. **추출 옵션 확인**
   - JS 문자열 ✓ (씬 안의 대사·내레이션)
   - HTML 본문 ✓ (UI 텍스트)
   - placeholder/title (선택)

3. **키 이름 패턴 입력**
   `ep01_taejo` 처럼 입력하면 `ep01_taejo_001`, `ep01_taejo_002` ... 식으로 키가 생성됨

4. **[텍스트 추출하기] 버튼 클릭**

5. **결과 확인**
   - **텍스트 목록** 탭: 모든 추출 텍스트를 한눈에 — *어색한 단어 찾기 좋음*
   - **JSON 출력** 탭: 사전 형태로 정리된 결과
   - **치환된 HTML** 탭: 원본에서 텍스트가 `{{ep01_taejo_001}}` 같은 ID로 바뀐 버전

6. **다운로드**
   - `texts.json`: 텍스트 사전
   - `replaced.html`: ID로 치환된 게임

7. **이후 작업**
   - `texts.json`만 다듬기 (단어 하나 바꾸기 = 2초)
   - 치환된 HTML은 게임 엔진이 키 → 텍스트로 다시 연결

## ❓ FAQ

**Q. 폴더가 이미 있는데 다시 실행하면?**
A. 안전해요. 이미 있는 파일은 절대 덮어쓰지 않습니다. 빠진 파일만 채워줘요.

**Q. tagline을 다른 걸로 바꾸고 싶어요.**
A. `kings.json`에서 해당 왕의 `tagline` 값을 수정하고 스크립트를 다시 돌리면 새 폴더의 파일만 갱신돼요. 기존 폴더에 이미 있는 파일은 안 바뀝니다 (보호).

**Q. ep28을 추가하고 싶어요.**
A. 조선 왕은 27명이지만, 폐세자나 추존왕(덕종, 원종, 진종, 장조, 문조 등)을 추가하고 싶다면 `kings.json`에 ep28~ 형태로 추가하면 됩니다.

**Q. 추출기로 뽑은 게 좀 이상해요. 너무 많거나 너무 적어요.**
A. 추출 옵션의 체크박스를 조정해보세요. JS 문자열만 켜고 다시 돌리면 게임 콘텐츠 위주로 잡혀요. HTML 본문까지 켜면 UI 텍스트도 다 잡혀서 양이 많아집니다.
